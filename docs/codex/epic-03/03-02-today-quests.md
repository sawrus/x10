# Today's quests widget

## Эпик

## Фича

Игровой день

## Контекст

Главный игровой экран должен показывать задания текущего дня.

## Задача для Codex

Создай widget Today'sQuests: группировка задач на доступные сегодня, выполненные сегодня, недоступные. Используй executions для определения выполненности.

## Используемое API

- `GET /api/v2/profiles/{profile_id}/tasks`
- `GET /api/v2/profiles/{profile_id}/executions`


## Ожидаемые файлы/зоны изменений
