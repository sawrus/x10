# UI Guidelines

## Тон продукта

X10 — поддерживающая Life RPG, а не обвиняющий todo-list.

Плохо:

- "Ты провалил день"
- "Ты не справился"
- "Штраф за лень"

Хорошо:

- "День закрыт. Завтра можно усилить результат."
- "Есть зона роста."
- "Минус к балансу — это честная обратная связь."

## Визуальный стиль

- Карточки задач должны выглядеть как квесты.
- Профиль должен выглядеть как персонаж.
- Сферы жизни должны выглядеть как характеристики/зоны развития.
- Негативные задачи должны отличаться визуально, но не выглядеть токсично.

## Основные экраны

- Dashboard
- Quest Board
- Character
- History
- Onboarding
