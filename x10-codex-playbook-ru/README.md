# X10 Codex Playbook RU

Этот пакет — набор готовых Markdown-промптов для Codex, чтобы последовательно разработать веб-игру **X10 Life RPG**.

## Как использовать

1. Открой следующий файл из `docs/codex/epic-XX/`.
2. Скопируй весь текст файла в Codex.
3. Дождись изменений.
4. Проверь сборку и тесты.
5. Закоммить изменения отдельным PR/commit.
6. Обнови `PROGRESS.md`.

## Главный принцип

Один prompt = один маленький PR.

Не проси Codex сделать весь эпик сразу. Двигайся строго по порядку.

## Порядок эпиков

1. `epic-00` — архитектура фронтенда
2. `epic-01` — основа игры
3. `epic-02` — первый игровой цикл
4. `epic-03` — Life RPG цикл
5. `epic-04` — игровой мир и Phaser
6. `epic-05` — удержание, контент и полировка

## Важные файлы

- `api-reference.md` — реальные API endpoints из `src/api/routes.rs`
- `00-development-rules.md` — правила работы Codex
- `01-frontend-architecture.md` — архитектура frontend
- `02-coding-rules.md` — правила кода
- `03-ui-guidelines.md` — UI/UX правила
- `PROGRESS.md` — прогресс выполнения
