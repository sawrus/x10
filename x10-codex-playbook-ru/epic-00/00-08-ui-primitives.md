# UI primitives

## Эпик

Epic 00

## Фича

Frontend Skeleton

## Контекст

Нужны базовые UI элементы.

## Задача для Codex

Создай базовые компоненты Button, Card, Input, Select, Modal, Badge, ProgressBar. Используй Tailwind. Компоненты должны быть простыми и переиспользуемыми.


## Ожидаемые файлы/зоны изменений

- `src/shared/ui`


## Что НЕ делать

- Не переписывать backend без прямой необходимости.
- Не придумывать новые API endpoints, если можно использовать существующие из api-reference.md.
- Не помещать бизнес-логику в Phaser.
- Не делать большую соседнюю фичу в рамках этого prompt.
